## Introduction

EZCARD is a software solution developed by Redmer Software Company for managing school pictures and camera cards. This application allows users to batch process images, manage student data, and print ID cards efficiently.

## Features

- Batch processing of images
- Management of student data
- Printing of ID cards
- Preferences for image storage and rotation
- Integration with Microsoft Access for data storage

## Installation

To install and run the EZCARD software, follow these steps:

1. Ensure you have Visual Basic 6.0 installed on your machine.
2. Clone the repository to your local machine:
   ```sh
   git clone https://github.com/yourusername/museum-redmer-software-ezcard.git
   ```
3. Open the `EZ_CARD.vbp` project file in Visual Basic 6.0.
4. Build the project to generate the executable.

## Usage

1. Launch the `EZ_CARD.exe` application.
2. Use the main form to manage batches of student images.
3. Use the preferences form to set image storage paths and rotation angles.
4. Print ID cards using the print functionality.
